# DOCX Report

Create structured DOCX reports, weekly status documents, and project summaries.

## Core Workflow

1. Confirm the report purpose, audience, date range, and required sections.
2. Use `references/report-outline.md` to structure the document.
3. Keep tables compact and make every status item include owner, date, and next action.
